Las máquinas en las que se trabajó son las 37, 38, 39 y 40.
Se encuentra la última versión en la carpeta /home/t3/Tarea_3-SD
en cada máquina.

De todas formas se incluyen todos los archivos necesarios para la ejecución, además, se puede
copiar el repositorio con:
git clone https://github.com/rettke/Tarea_3-SD

Para la ejecución se debe asegurar de compilar los archivos en la carpeta raíz con:
make

Luego en la máquina respectiva se debe añadir el número de la máquina para
ejecutar respectivamente con:
make 37
make 38
make 39
make 40
